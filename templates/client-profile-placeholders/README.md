# Client Profile Placeholders

These templates are public-safe examples. They are not working profiles.

Use placeholders only:

- `EXAMPLE_HOSTNAME`
- `EXAMPLE_IP`
- `EXAMPLE_PATH`
- `EXAMPLE_UUID`
- `EXAMPLE_PUBLIC_KEY`
- `EXAMPLE_SHORT_ID`
- `EXAMPLE_SNI`
- `EXAMPLE_HOST_HEADER`
- `EXAMPLE_CLIENT_PROFILE`

Do not commit active client import links or QR codes.
