# No Tor Scope Note

This repository does not provide Tor setup, Tor bridge setup, Tor-dependent workflows, or Tor transport instructions.

Tor-related systems may appear only as a neutral related-technology reference when explaining the broader censorship-resistance landscape. They are not recommendations, templates, or implementation paths in this repo.

Reasoning:

- The source project explicitly excluded Tor-dependent workflows.
- This repository focuses on route families that can be discussed without publishing live private infrastructure.
- Keeping scope narrow reduces confusion for non-technical users and contributors.
